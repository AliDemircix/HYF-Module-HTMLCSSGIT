
1. How do I create a hidden file or folder? How do I display it in the CLI?

Answer 1 is :	mkdir .HiddenFolderName
		ls -a

2. How do I create multiple nested directories, like /c/Users/myusername/these/folders/are/just/for/fun?

Answer 2 is :	mkdir -p this/directory/is/my/multiple/nested/directories
	

3. How do I append a message to a file, without a newline character?

Answer 3 is :	echo "first message" >> testfile.txt
		echo "second message" >> testfile.txt
